Računalne vještine

  • C#
    • Programski jezik koji se najčešće koristi kod razvijanja aplikacija i računalnih igrica
  • C++
    • Programski jezik koji se zna naći kod razvijanja računalnih igrica
  • HTML
    • HTML (Hypertext Markup Language) je standardni jezik za strukturiranje i predstavljanje sadržaja na webu
  • CSS
    • CSS (Cascading Style Sheets) je stilski jezik koji se koristi za definiranje izgleda i dizajna web stranica koje su napisane u HTML-u
  • Angular
    • Angular je popularni open-source okvir za izgradnju web aplikacija razvijen od strane Googlea. To je TypeScript-baziran okvir koji se koristi za izgradnju dinamičnih i skalabilnih aplikacija s bogatim korisničkim sučeljem.
  • .NET
    • .NET je razvojni okvir koji je razvio Microsoft za izgradnju raznovrsnih aplikacija, uključujući web, desktop, mobilne i IoT aplikacije. On je podržan na različitim platformama, kao što su Windows, Linux i macOS.
  • SQL
    • SQL (Structured Query Language) je standardni jezik za upravljanje i manipulaciju relacijskim bazama podataka. SQL omogućuje korisnicima da definiraju, upituju, uređuju i upravljaju podacima u bazama podataka.
  • Unity
    • Unity je popularan i moćan razvojni okvir za izradu igara, simulacija i interaktivnih aplikacija. On podržava više platformi, uključujući računalne igre, konzole, mobilne uređaje i virtualnu stvarnost. Unity pruža intuitivno korisničko sučelje i bogat set alata za razvoj igara. On podržava scripting jezike kao što su C# i UnityScript (sličan JavaScriptu), omogućavajući programerima da implementiraju funkcionalnosti igre i interakcije.
  • JavaScript
    • JavaScript je visokonivo programski jezik koji se široko koristi za izgradnju dinamičnih web stranica i web aplikacija. On je standardni jezik za programiranje na klijentskoj strani, što znači da se izvršava u web pregledniku korisnika. JavaScript omogućuje interaktivnost na web stranicama, kao što su animacije, promjena sadržaja stranice u realnom vremenu, validacija formi i interakcija s korisnicima. Također se koristi za izgradnju složenijih aplikacija kao što su web aplikacije, igre i mobilne aplikacije.
  • PHP
    • PHP (Hypertext Preprocessor) je popularan skriptni jezik za web razvoj. On se koristi za izgradnju dinamičnih web stranica i web aplikacija. PHP se izvršava na serveru, što znači da generira HTML kod koji se zatim šalje klijentskom pregledniku. PHP je vrlo fleksibilan jezik koji podržava različite stilove programiranja, uključujući proceduralno i objektno orijentirano programiranje. On ima bogat skup funkcija i biblioteka koji olakšavaju rad s bazama podataka, manipulaciju datotekama, obradu formi, slanje e-pošte i još mnogo toga.
